・ What is TextViewerPlugin ps2txt for PogoShell2?

A text browsing tool for Game Boy Advance that runs on PogoShell.
I think that you can understand the feeling even if you start up the EMU alone, so please try it.
The attached sample plug-in contains only the first level kanji to reduce the capacity. I think it's not so difficult, so please recreate the font if possible.

・ How to install.

Copy txt.bin under PogoShell2 \ root \ .shell \ plugins \.
Open PogoShell2 \ root \ .shell \ pogo.cfg with Notepad and rewrite the line of
txt 4 TXT
to
txt 4 txt.bin 2
. So that you can put it back
#txt 4 TXT
txt 4 txt.bin 2
maybe better.
